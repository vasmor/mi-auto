# MI-AUTO — Разбор импорта демо-контента

## 1. Механизм запуска

| Параметр | Значение |
|----------|----------|
| URL запуска | `/wp-admin/?miauto_setup=1` |
| Файл | `inc/demo-import/demo-import.php` |
| Хук | `admin_init` |
| Требуемые права | `manage_options` |
| Одноразовость | Флаг `miauto_demo_imported` в `wp_options` |
| Метод | Программный (WordPress API), без XML/WXR файлов |

### Последовательность шагов

1. Загрузка изображений из `img/` в медиатеку
2. Создание формы Contact Form 7
3. Установка Theme Options (Carbon Fields)
4. Создание страниц (7 шт.)
5. Настройки чтения (статическая главная)
6. Создание записей CPT (22 шт.)
7. Заполнение мета-полей страниц
8. Создание блог-записей (3 шт.)
9. Создание навигационного меню
10. Настройка Permalinks

## 3. Контактная форма CF7

| Параметр | Значение |
|----------|----------|
| Название | Запись на обслуживание |
| Метка | `_miauto_demo_form = '1'` |

**Поля формы:**
- `your-name` (text, обязательное) — Ваше имя
- `your-phone` (tel, обязательное) — Телефон
- `your-service` (select) — Выберите услугу / ТО / Ремонт двигателя / Ремонт подвески / Диагностика / Другое
- submit — Записаться

**Email:**
- Тема: `MI-AUTO: Новая заявка от [your-name]`
- Получатель: admin_email из настроек WP
- Тело: Имя, Телефон, Услуга

## 4. Настройки WordPress

### Чтение
| Опция | Значение |
|-------|----------|
| `show_on_front` | `page` (статическая главная) |
| `page_on_front` | ID страницы «Главная» |
| `page_for_posts` | ID страницы «Блог» |

### Permalinks
- Структура: `/%postname%/`
- `flush_rules()` вызывается автоматически

### Меню
- Локация `primary` → меню «Главное меню»

### Флаги
- `miauto_demo_imported` = `true` (предотвращает повторный импорт)
